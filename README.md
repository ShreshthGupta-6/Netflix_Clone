# Netflix_Clone
This repository consists of a responsive clone of the Netflix website.

## 🚀 Features
* **Responsive Design:** Fully optimized for fluid layouts across mobile, tablet, and desktop screens.
* **Sleek UI:** Mimics the exact look and feel of Netflix, including the iconic hero banner, movie rows, and navigation.
* **Interactive UI Elements:** Features smooth CSS transitions and hover effects.

## 📦 Getting Started
To view and run this project locally, follow these simple steps:
1. Clone the repository:
   git clone [https://github.com/ShreshthGupta-6/Netflix_Clone.git](https://github.com/ShreshthGupta-6/Netflix_Clone.git)
2. Navigate to the project directory:
cd Netflix_Clone
3. Open the project:
Simply double-click the index.html file, or use the Live Server extension in VS Code to launch it locally.
